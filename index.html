🔑 Plural Spelling Rules

🐱
+s → cat → cats, dog → dogs
🚌
+es → bus → buses, box → boxes
👶
-y → -ies → baby → babies, city → cities
🔪
-f/-fe → -ves → knife → knives, leaf → leaves
🍅
-o → -es → tomato → tomatoes, potato → potatoes
